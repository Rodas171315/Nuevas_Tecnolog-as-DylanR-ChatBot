TacoBotProject

El bot para buscar empleo e interactuar con reclutadores.

Visita TacoBot en Facebook para empezar a interactuar.

Tecnología

TacoBot está escrito en C# y se conecta a una instancia de DataBase para guardar oportunidades de trabajo.

Utiliza el Microsoft Bot Framework de código abierto para crear diálogos y establecer las conexiones con un interpretador de lenguaje natural como LUIS y QnA Maker.

La estructura de TacoBot se encuentra completamente alojada en Azure, que ofrece un nivel gratuito y permite interactuar con casi cualquier aplicación de mensajería.

Cuando está en ejecucion, actúa conforme a una Web de Azure, mientras que en un entorno local actúa con el Microsoft Bot Framework Emulator donde puede hablar con TacoBot desde el emulador y guardar los datos en una instancia local del DB.
